Guía del Operador

¿Qué es este sistema?

Este es un sistema de trading cuántico avanzado que utiliza algoritmos de inteligencia artificial para analizar el mercado de criptomonedas en tiempo real. El sistema procesa datos de Binance y genera señales de trading basadas en análisis cuántico.

Cómo interpretar las métricas

Puntuación Cuántica: Valor entre 0-1 que indica la fuerza del análisis. Señales Activas: Número de oportunidades detectadas. Coherencia Cuántica: Estabilidad del sistema (ideal >60%).

Señales de Trading

NEUTRAL: Estrategias de opciones (straddle/strangle). BUY/SELL: Direcciones claras con alta confianza. Confianza: Porcentaje de certeza del algoritmo (>70% = alta confianza).

Gestión de Riesgos

Nunca opere con más del 2% del capital por señal
Use stop-loss automáticos
Diversifique entre múltiples activos
Monitoree la coherencia cuántica constantemente

Configuración para Trading Real

1. Edite el archivo .env con sus claves API de Binance
2. Configure BINANCE_TESTNET=false para modo producción
3. Reinicie el sistema con node frontend-api.js
